# Loxley AI — Corridor Intelligence Portal

Subscriber-gated intelligence dashboard for Watkins-scored real estate corridors.

## Dev

Open `index.html` directly or serve locally:

```bash
python3 -m http.server 3000
```

## Deploy

Deployed via [Vercel](https://vercel.com) at [loxleyai-portal.vercel.app](https://loxleyai-portal.vercel.app).
